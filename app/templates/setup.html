For now, please edit the config files at these location in container:

  • /root/.chia/plotman/plotman.yaml
  • /root/.chia/mainnet/config/config.yaml

This is likely mapped to a folder on your Docker host like this example on Unraid:

    /mnt/user/appdata/machinaris/plotman/plotman.yaml /mnt/user/appdata/machinaris/mainnet/config/config.yaml

To get into the container, you can execute this or similar:

docker exec -it machinaris bash
